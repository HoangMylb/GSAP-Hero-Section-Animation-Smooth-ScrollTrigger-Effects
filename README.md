# GSAP-Hero-Section-Animation-Smooth-ScrollTrigger-Effects
A hero section built using GSAP and ScrollTrigger to create smooth scrolling animations, parallax effects, and interactive motion. This demo includes:  ✔ ScrollTrigger-based animations ✔ Parallax movement ✔ Text reveal effects ✔ Optimized performance (60 FPS) ✔ Fully responsive layout  Tools: HTML, CSS, JavaScript, GSAP, ScrollTrigger
